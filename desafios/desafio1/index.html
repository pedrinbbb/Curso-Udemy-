O HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

O CSS

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de um documento escrito em HTML. Ele permite controlar a aparência visual dos elementos da página, como cores, fontes, espaçamentos e layouts.

O JavaScript

JavaScript é uma linguagem de programação interpretada e dinâmica, usada principalmente para adicionar interatividade a páginas web. Ela permite manipular elementos HTML e CSS, responder a eventos do usuário e criar aplicações web completas.
